# Polynomial Regression on Social Network Ads

Predicts **Estimated Salary** from **Age** using Polynomial Regression (degree=2).

## Dataset
Social Network Ads dataset — 400 rows, 5 columns  
Features used: `Age` (input) → `EstimatedSalary` (target)

## What the notebook covers
1. Load and explore the dataset
2. Check for nulls and duplicates
3. Select feature (Age) and target (EstimatedSalary)
4. Apply Polynomial Features transformation (degree=2)
5. Train Linear Regression on transformed features
6. Predict and visualize the polynomial curve

## Why Polynomial Regression?
Simple Linear Regression fits a straight line.  
Polynomial Regression fits a **curve** — better when data has a non-linear pattern.

## How to run
1. Install dependencies: `pip install -r requirements.txt`
2. Open `Social_Network_Ads_explained.ipynb` in Jupyter
3. Run all cells
